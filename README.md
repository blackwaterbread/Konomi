# Konomi

[한국어](https://github.com/blackwaterbread/Konomi/blob/main/README.ko.md)

Konomi is an Electron desktop app that scans user-selected folders, extracts image metadata, helps you search and organize your archive, and lets you generate new images with NovelAI in the same place. Rather than being just a viewer, it is designed to unify the full workflow of "generate -> archive -> search -> reuse."

## Key Features

- Local folder-based image management
- Compatibility with NovelAI and WebUI PNG metadata
- Search by prompt, tag, resolution, and model
- Favorites, custom categories, and random pick
- Duplicate file detection and duplicate-resolution flow when adding folders
- Hybrid similar-image analysis based on perceptual hash and Jaccard similarity
- Prompt editing that treats prompts like reusable building blocks
- Reusable prompt group system
- Advanced and conditional prompt features
- Image generation through the NovelAI API
- Automatic image generation
